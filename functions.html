What are Functions?

Functions are blocks of code designed to perform a specific task. In Programming Language X, you can define a function like this: 


function myFunction() {
    // code to be executed
    return "Hello, World!";
}

This declares a function called myFunction that returns the string "Hello, World!" when called.

Examples of Function Usage:

  • myFunction(); - Calls the function and returns "Hello, World!"
  • function add(a, b) { return a + b; } - A function that adds two numbers
