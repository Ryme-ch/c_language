What are Loops?

Loops are used to execute a block of code multiple times. In Programming Language X, the most common loops are for, while, and do-while.

For Loop Example:


for (var i = 0; i < 5; i++) {
    console.log(i);
}

This loop runs 5 times, printing numbers from 0 to 4.

While Loop Example:


var count = 0;
while (count < 5) {
    console.log(count);
    count++;
}

This loop also runs 5 times, similar to the for loop.

Do-While Loop Example:


var count = 0;
do {
    console.log(count);
    count++;
} while (count < 5);

The do-while loop ensures the code is executed at least once before checking the condition.
