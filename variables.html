What are Variables?

Variables are used to store data values. In Programming Language X, you can declare a variable like this: 

var x = 10;

This declares a variable x and assigns it the value of 10.

Examples of Variable Usage:

  • var name = "John"; - A string variable
  • var isCompleted = true; - A boolean variable
  • var score = 99.5; - A float variable
